## Bill Of Materials:
- 1x Seeed Studio XIAO RP2040
- 1x EC11 rotary encoder
- 1x 0.91" 128x32 OLED display
- 8x MX-style switches
- 8x White blank DSA keycaps
- 8x Through-hole 1N4148 diodes 
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
